Excel Sales Dashboard Analysis

Project Overview

This project analyzes sales data and presents key business insights through an interactive Excel dashboard.

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- XLOOKUP
- SUMIFS

Objectives

- Analyze sales trends over time
- Identify top-performing product categories
- Compare sales across countries
- Track order status
- Build an interactive dashboard for decision-making

Key Insights

- Sales increased significantly during Q4
- A small group of customers generated most of the revenue
- Some high-sales categories had lower profit margins
- The USA contributed the highest sales volume

Dashboard Features

- Yearly Sales Analysis
- Monthly Sales Trends
- Product Status Breakdown
- Country-Wise Sales
- Product Line Distribution
- Interactive Filters

Files Included

- Raw dataset
- Excel dashboard
- Dashboard screenshot
